# ELK Stack for Text Mining

It creates a ELK Stack using Docker to mining texts.

# Requirements
- [Docker](https://docker.com) **v20+**
- [Makefile](https://pt.wikibooks.org/wiki/Programar_em_C/Makefiles)

# Using
- Logstash image **v8.6**

# Commands

## Run

```sh
make run
```
It'll run in an interactive mode.

## Stop
```sh
CTRL+c
```
